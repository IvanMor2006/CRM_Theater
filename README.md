# CRM для управления театром

Десктопное приложение для автоматизации работы театра. Позволяет управлять спектаклями, представлениями, билетами, сотрудниками, ролями и исполнителями.

## Возможности

- **Управление 11 таблицами**: билеты, представления, спектакли, залы, роли, исполнители, сотрудники, должности, пьесы, жанры
- **CRUD-операции**: добавление, редактирование и удаление записей с проверкой целостности данных
- **Динамический интерфейс**: таблицы и формы создаются автоматически на основе конфигурации
- **Вложенные формы**: выбор связанных записей (спектакль для роли, сотрудник для исполнителя)
- **Обработка ограничений**: понятные сообщения при нарушении уникальности, проверки дат, отрицательных цен
- **Автоматическая ширина колонок**: подстраивается под содержимое

## Технологии

| Технология | Назначение |
|------------|------------|
| Python 3.10+ | Язык программирования |
| Tkinter | Графический интерфейс |
| pyodbc | Подключение к SQL Server |
| SQL Server Express | База данных |

## Установка и запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```
### 2. Настройте базу данных

Установите SQL Server Express
Создайте базу данных с именем Театр
Выполните скрипт создания таблиц (LR8.sql)
Убедитесь, что сервер запущен на localhost\SQLEXPRESS01

### 3. Запустите приложение

```bash
python window.py
```
